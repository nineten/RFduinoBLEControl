# RFduinoBLEControl

iOS version of a BLE Central Core control for example RFduinoBLE modular stuff.

This app sends commands to those modules using BLE.


## Setup

* Clone code.
* Open in Xcode.
* Run on iOS device with iOS 7.0 and above.
